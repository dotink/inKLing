Welcome to inKLing

This page is an example of what inKling can do with essentially no custom CSS. The entirity of custom CSS on this page is roughly 10 - 20 lines and consists mostly of what is needed to define some colors for simple visualization and our very basic nav style (in our case 4 nav items of equal size taking up full width with centered text - 3 lines).

This page uses some nwer HTML5 tags, however, inKLing provides full support for using HTML tag names as classes on divs for all its major functionality.

Headings Are Normalized

You can see from the grouped headings and paragraphs below that all headings consume the same height, however, provide varying degreess of top and bottom margin.
